# SQL Exploration and Advanced Analytics

The data used in this analysis originates from the previously developed Data Warehouse project, where raw data was processed through the Medallion Architecture (Bronze → Silver → Gold layers).

For analytical purposes, the Gold layer tables were used, which contain cleaned and business-ready datasets optimized for reporting and advanced SQL analysis.

Primary tables used:

* gold.fact_sales

* gold.dim_customers

* gold.dim_products

This section demonstrates how SQL can be applied to perform data exploration, business intelligence analysis, and advanced analytical reporting.
